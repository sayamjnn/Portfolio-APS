TreeNode Class for Binary Tree

The TreeNode class represents a node in a binary tree structure. Each node contains integer data and references to its left and right child nodes.

TreeNode Class in Java

public class TreeNode {
    int data;
    TreeNode left, right;

    public TreeNode(int data) {
        this.data = data;
        this.left = null;
        this.right = null;
    }
}

Usage

This class is used to construct binary trees where each node can have at most two children. It provides a basic structure for building, manipulating, and traversing binary trees in Java.
